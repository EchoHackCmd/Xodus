Download: https://github.com/EchoHackCmd/Xodus/releases/download/1.4/Xodus-1.4.exe

- G: Survival
- L: Phase
- V: Air Acceleration Speed
- K: AntiKnockBack
- I: AntiWeb
- X: Auto Sprint
- C: NoSlowDown
- B: Aimbot

- ">": Module Value Increase
